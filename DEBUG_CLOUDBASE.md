## 常见入口位置

根据你的描述（有"服务管理"和"环境设置"），尝试以下路径：

### 路径 A：应用管理
```
左侧菜单
    └── 应用管理（或"应用"）
            ├── 我的应用
            │       └── [创建应用] 按钮
            └── 云托管应用
                    └── [新建服务] 按钮
```

### 路径 B：云服务
```
左侧菜单
    └── 云服务（或"基础服务"）
            ├── 云函数
            ├── 云数据库
            └── 云托管
                    └── [立即开通] 或 [创建服务]
```

### 路径 C：开发
```
左侧菜单
    └── 开发
            ├── 开发设置
            └── 云服务
                    └── 云托管
```

### 路径 D：直接搜索
```
页面顶部搜索框
    └── 输入"云托管" 或 "CloudRun"
```
