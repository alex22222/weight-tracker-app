## 第一步：购买域名（预计 10 分钟）

### 操作地址
https://dnspod.cloud.tencent.com

### 操作步骤

1. 登录腾讯云账号
2. 在搜索框输入：`weight-tracker`
3. 选择后缀 `.cn`（¥35/年）
4. 点击「查询」查看可用性
5. 如果可用，点击「加入购物车」
6. 如果不可用，尝试以下备选：
   - `weight-tracker-app.cn`
   - `my-weight-tracker.cn`
   - `weighttracker.cn`
   - `ranzhi-club.cn`
   - `fit-track.cn`

7. 去购物车结算
8. 完成支付

---

## 第二步：实名认证（预计 30 分钟）

### 操作路径
腾讯云控制台 → 域名注册 → 域名管理 → 实名认证

### 所需材料
- 身份证正反面清晰照片
- 本人实名认证的手机号

### 操作步骤
1. 点击「立即认证」
2. 选择认证类型：个人
3. 填写真实姓名和身份证号
4. 上传身份证正反面照片
5. 提交审核
6. 等待审核通过（通常 1-24 小时）

---

## 第三步：域名备案（预计 10-25 天）

### 操作地址
https://cloud.tencent.com/product/ba

### 备案前准备
| 材料 | 要求 |
|------|------|
| 身份证 | 清晰照片，剩余有效期 > 3 个月 |
| 手机号 | 本人实名手机号 |
| 紧急联系人 | 姓名+电话，不能与本人相同 |
| 人脸识别 | 下载腾讯云APP |

### 操作步骤

1. 点击「开始备案」→「新增备案」

2. 填写主体信息：
   ```
   主体类型：个人
   姓名：你的真实姓名
   证件类型：身份证
   证件号码：你的身份证号
   联系方式：你的手机号
   电子邮箱：你的邮箱
   通讯地址：你的详细地址
   ```

3. 填写网站信息：
   ```
   域名：weight-tracker.cn（你购买的域名）
   网站名称：燃脂俱乐部
   网站首页：weight-tracker.cn
   网站服务内容：其他
   网站语言：中文
   网站备注：个人体重管理小程序，用于记录体重和健身数据
   ```

4. 上传材料：
   - 身份证正反面
   - 本人手持身份证照片

5. 下载腾讯云 APP，完成人脸识别

6. 提交审核

### 审核流程
```
腾讯云初审（1-2个工作日）
    ↓
短信核验（管局发送验证码）
    ↓
管局审核（7-20个工作日）
    ↓
备案成功！
```

## 第四步：配置 CloudBase 自定义域名

### 操作步骤

1. 登录 CloudBase 控制台
   https://console.cloud.tencent.com/tcb

2. 进入环境：`weight-tracker-1ghr085dd7d6cff2`

3. 点击「云托管」→ 找到 `weight-tracker-api` 服务

4. 点击「服务设置」→「自定义域名」→「添加域名」

5. 输入子域名：
   ```
   api.weight-tracker.cn（替换为你的域名）
   ```

6. 复制 CloudBase 提供的 CNAME 值

7. 前往腾讯云 DNSPod 控制台配置解析：
   ```
   记录类型：CNAME
   主机记录：api
   记录值：[CloudBase 提供的域名]
   TTL：默认
   ```

8. 等待 SSL 证书自动申请（5-10分钟）

## 🆘 常见问题

### Q1: 备案期间可以用吗？
A: 不可以。备案期间网站必须关闭，等备案通过后才能正式上线。

### Q2: 可以用香港/海外服务器免备案吗？
A: 可以，但 CloudBase 是国内服务，必须使用国内服务器，所以必须备案。

### Q3: 备案被拒绝了怎么办？
A: 根据拒绝原因修改后重新提交。常见原因：照片不清晰、网站名称不合规。

### Q4: 域名必须买 .cn 吗？
A: 不一定。.com/.net 也可以，价格稍贵一点。
